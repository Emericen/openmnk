# OpenMNK (v0 — Legacy)

> **This is the earlier open-source version of OpenMNK.** It uses screen capture with mouse and keyboard control to operate the computer. The latest version has moved to a terminal + AppleScript architecture, which is simpler and more reliable. See [openmnk.com](https://openmnk.com) for the latest.

---

Open-source desktop tool that lets LLM see your screen and use mouse and keyboard. Plug in any OpenAI-compatible model and let it operate your computer.

<p align="center"><img src="assets/banner.png" alt="OpenMNK demo" /></p>

### Quickstart

```bash
git clone https://github.com/Emericen/openmnk-legacy.git
cd openmnk-legacy
cp .env.example .env         # add your API key
npm install
npm run dev
```
